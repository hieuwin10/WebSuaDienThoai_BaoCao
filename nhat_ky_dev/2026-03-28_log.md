# Nhật ký phát triển - 28/03/2026

## 1. Công việc đã hoàn thành hôm nay
Hôm nay chúng ta đã tập trung vào việc xây dựng và hoàn thiện lớp Frontend (React + Vite) và tích hợp sâu với Backend để đảm bảo hệ thống vận hành đúng luồng nghiệp vụ.

### 🔹 Frontend (Giao diện & Logic)
- **Tích hợp API:** Hoàn thiện kết nối cho hơn 8 model chính: Auth, Profile, Devices, Services, Components, Repair Tickets, Warranties và Dashboard.
- **Giao diện Premium:** Sử dụng hệ thống thiết kế Dark Mode hiện đại, đồng bộ hóa font chữ "Outfit" và hệ thống icon Lucide.
- **Dashboard:** Triển khai biểu đồ thống kê bằng Recharts hiển thị doanh thu và đơn hàng.
- **Module Sửa chữa:** Hoàn thiện luồng: Tạo thiết bị -> Chọn dịch vụ -> Sử dụng linh kiện -> Tự động trừ kho -> Tạo phiếu bảo hành.

### 🔹 Backend & Tối ưu hóa API
- **Xử lý Xung đột:** Chuyển toàn bộ hệ thống sang cổng mới (Backend: `5001`, Frontend: `5175`) để tránh lỗi hệ thống.
- **Sửa lỗi Status Code:** Thay đổi lỗi 404 nhầm lẫn sang 400 (Bad Request) và 401 (Unauthorized) cho các lỗi Validation và Auth.
- **Tương thích MongoDB:** Tạm thời tắt cơ chế Transactions để đảm bảo dev local không cần Replica Set, giúp quá trình phát triển mượt mà hơn.
- **Cấu hình Interceptors:** Axios tự động đính kèm `Authorization: Bearer <token>` và xử lý redirect khi hết hạn login.
